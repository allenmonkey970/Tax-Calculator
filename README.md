# Tax Calculator

This repository contains a Python script that calculates the tax owed based on the user's annual salary and the current tax brackets in the United States. The script accounts for standard deductions and applies the appropriate tax rates to determine the final tax amount.

## Description

The Tax Calculator helps users to quickly determine their taxable salary and the amount of federal tax they owe. The calculator takes into consideration the standard tax deduction and applies the current tax rates to different income brackets. 
## Features

- Input: User's gross annual salary
- Output: Taxable salary and federal tax owed
- Adjustable tax brackets and rates
- Error handling for non-numeric input

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/allenmonkey970/Tax-Calculator.git
    ```
2. Navigate to the project directory:
    ```bash
    cd Tax-Calculator
    ```
3. Run the script:
    ```bash
    python MathTaxes.py
    ```

## Usage

1. The script will prompt you to enter your gross annual salary.
2. It will calculate your taxable salary by deducting the standard deduction.
3. The script will then compute and display the tax owed based on current tax brackets.

```bash
python MathTaxes.py
```

Example output:
```
How much do you make a year? 50000
Taxable Salary: $35,400.00
Tax owed: $4,074.00
```
